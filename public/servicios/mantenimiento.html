Cambio puntual o mantenimiento mensual

No es lo mismo hacer un ajuste aislado que mantener una web revisada con continuidad.

Cambio puntual

Un ajuste concreto.

Pensado para modificar algo específico sin contratar un seguimiento mensual.

  • Cambios de texto
  • Nuevas imágenes
  • Ajustes visuales
  • Nueva sección
  • Corrección de errores
  • Pequeñas mejoras

Mantenimiento mensual

Revisión y soporte continuo.

Pensado para negocios que quieren tener su web actualizada, revisada y preparada para seguir mejorando.

  • Cambios mensuales
  • Revisión técnica
  • Soporte
  • Ajustes visuales
  • Revisión de formularios
  • Pequeñas mejoras
  • Seguimiento básico

No todo negocio necesita mantenimiento mensual. Primero vemos si tiene sentido para tu web y para la forma en la que la usas.

Qué podemos mantener

Contenido

Textos, imágenes, secciones, servicios o información del negocio.

Formularios

Revisión de formularios, contacto, enlaces y avisos.

Diseño

Ajustes visuales para mantener la web cuidada y coherente.

Técnica

Revisión básica de errores, carga, enlaces y funcionamiento.

Nuevas secciones

Añadir bloques o páginas cuando el negocio lo necesite.

Mejoras continuas

Pequeños cambios para que la web siga evolucionando.
